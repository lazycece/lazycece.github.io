springboot使用swagger2生产api文档

Posted on 2019-01-31 Edited on 2021-01-31 In springboot Views:

日常开发中，如果对接的是终端（前端、iOS、Android、PC等)，那就不仅仅是需要完成接口的编码，编写接口文档也同样很重要，因为一份完善的接口文档可以节省后端开发大量和写终端人的撕逼时间，减少很多不必要的麻烦。当然，如果自己一个人把前后端事情的干了，那就另当别论了。

能够写一份好的接口文档很重要，能够高效率的写出且后续易于维护更重要，因为谁也不想把时间浪费在这种繁琐无味的事情上。swagger能帮助我们做到这一点，它是自动生成接口文档，节约了时间，还便于维护。下面介绍一下swagger2在springboot中的应用：
