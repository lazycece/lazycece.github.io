解决方案

在使用的时候可以选择如下任意一种方式：

  • 在创建MongoClient实例的时候设置pojoCodecRegistry

  • 在使用MongoDatabase实例的时候设置pojoCodecRegistry

  • 在使用MongoCollection实例的时候设置pojoCodecRegistry
