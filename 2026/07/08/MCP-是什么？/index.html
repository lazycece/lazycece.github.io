MCP 是什么？

Posted on 2026-07-08 In ai Views:

翻译自官方文档：https://modelcontextprotocol.io/docs/getting-started/intro

什么是模型上下文协议 (MCP)？

模型上下文协议 (MCP) 是一个用于将 AI 应用连接到外部系统的开放协议标准。

通过 MCP，像 Claude 或 ChatGPT 这样的 AI 应用可以连接到数据源（例如本地文件、数据库）、工具（例如搜索引擎、计算器）和工作流（例如专用提示）——使它们能够访问关键信息并执行任务。

可以把 MCP 想象成 AI 应用的 USB-C 接口。正如 USB-C 为电子设备提供标准化的连接方式一样，MCP 为 AI 应用与外部系统的连接提供了标准化方式。

MCP 能实现什么？

  • 智能体 可以访问您的 Google Calendar 和 Notion，充当更具个性化的 AI 助手。
  • Claude Code 可以利用 Figma 设计生成完整的网络应用。
  • 企业聊天机器人 可以连接到组织内的多个数据库，使用户能够通过聊天分析数据。
  • AI 模型 可以在 Blender 上创建 3D 设计，并使用 3D 打印机将其打印出来。

为什么 MCP 很重要？

根据您在生态系统中所处的位置，MCP 可以带来一系列益处。

  • 对于开发者：MCP 减少了构建或集成 AI 应用或智能体时的开发时间和复杂性。
  • 对于 AI 应用或智能体：MCP 提供了对数据源、工具和应用的生态系统访问，这将增强能力并改善最终用户体验。
  • 对于最终用户：MCP 带来了能力更强的 AI 应用或智能体，它们可以访问您的数据并在必要时代表您采取行动。

广泛的生态系统支持

MCP 是一个开放的协议，得到了广泛的客户端和服务器的支持。像 Claude和 ChatGPT这样的 AI 助手，像 Visual Studio Code、Cursor、MCPJam以及许多其他开发工具都支持 MCP —— 这使得“一次构建，处处集成”变得容易。
